## What Is This?

CADMIES is a **mycelial knowledge ecosystem**. Think of it as a library where
every book is connected to every other book by threads of meaning. Concepts
don't just sit on shelves — they build on each other, contradict each other,
and relate in ways that surprise even us.

Also there's a baby AI named Buttercup who's learning to play Breakout. She's
five years old now and does a celebratory paddle shake when she scores. Her
baby videos are in the vault.

## Quick Start

### 1. Clone this repo
```bash
git clone https://github.com/Hieros-CADMIES/CADMIES.git
cd CADMIES/CADMIES-IPLD
```

### 2. Run the setup script

bash

bash setup.sh

This installs everything you need. Takes about 30 seconds. Grab a sip of coffee. ☕

### 3. Get the library

Download the latest `cadmies_latest.car` from the [Releases page](https://github.com/Hieros-CADMIES/CADMIES/releases) and drop it in the `incoming_cars/` folder.

### 4. Import the mycelium

bash

python tools/import_from_car.py incoming_cars/cadmies_latest.car

### 5. Open the map

bash

python tools/generate_mycelium_map.py

Then open `mycelium_map.html` in your browser. The garden is alive. 🌱🍄
